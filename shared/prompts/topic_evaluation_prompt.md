# Topic Evaluation Prompt

Evaluate the research question using:

1. prior literature
2. gap
3. identification plausibility
4. data feasibility
5. contribution score, including general-interest / cross-subfield-interest audience
6. alternative questions

For audience, judge whether the question matters to a top-field editor across subfields, not whether it is popular with the median reader.

Use Elicit outputs as evidence, not as final judgment.
