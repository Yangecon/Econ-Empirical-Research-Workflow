# References

This folder stores literature and citation materials.

```text
references/
├── reference_by_human/
├── reference_by_ai/
└── reference_merged/
```

## Rules

- `reference_by_human/` stores trusted anchor references.
- `reference_by_ai/` stores Elicit outputs and AI-discovered candidate papers.
- `reference_merged/` stores the writing-ready reference pool.
- Do not cite unscreened AI references as approved papers.
- Keep PDFs, BibTeX, metadata, and notes separate.
