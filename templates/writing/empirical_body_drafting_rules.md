# Empirical Paper Body Drafting Rules

Generic rules for English empirical economics writing.

## Data

Data explains data. It does not report results.

Use conceptual subsection names. Do not overload headings with data-source names or construction details.

## Empirical strategy

Write the main specification once. Define the outcome, treatment, timing, fixed effects, controls, standard errors, sample, and coefficient of interest.

## Results

Begin with the baseline specification or point back to where it is defined.

Explain economic meaning before reporting mechanical coefficient notation.

Use benchmarks such as baseline means, standard deviations, pre-period values, or economically relevant counterfactuals.

## Heterogeneity

Write the interaction specification once. Explain the main effect, interaction effect, omitted group, and economic interpretation.

## Additional outcomes

Do not repeat the full specification unless the empirical design changes.

## Multi-table narrative

Use related tables as one argument. State the theme first, then explain how each table advances the claim.

## Main text vs appendix

Main text keeps primary identification and conclusions.

Appendix gets:

- alternative fixed effects
- alternative clustering
- sample restrictions
- long data construction details
- long API or prompt details
- secondary robustness checks

## Abbreviations

Write out abbreviations on first use.
