# Empirical Paper Body Template

## Data

### <Conceptual Data Object 1>

Describe source, sample construction, unit of observation, coverage, variable definition, and measurement limitations. Do not report results.

### <Conceptual Data Object 2>

Describe the second core data object using the same logic.

## Empirical Strategy

Define the baseline estimating equation. Explain outcome, treatment or exposure, fixed effects, controls, standard errors, timing, sample, and coefficient of interest.

## Results

### Baseline Specification

Introduce or restate the preferred specification. Explain what the first result table tests.

### Baseline Results

Report the main estimate with economic magnitude and a meaningful benchmark.

### Heterogeneity

Define the interaction specification. Explain how to interpret the interaction coefficient and reference group.

### Additional Outcomes

Use the same specification unless the design changes. Do not repeat the full baseline equation.

### Mechanisms

Connect related tables or figures into one economic narrative.

### Robustness Summary

Summarize only the checks needed for main-text credibility. Put detailed checks in the appendix.
