# Multi-Table Narrative Stitching Template

## Goal

Use multiple tables as one argument rather than isolated result dumps.

## Generic template

```text
[Theme sentence.] Tables X through Z examine [economic theme]. Table X first shows [step 1]. Table Y then [step 2]. Table Z further clarifies [step 3]. Together, these results indicate that [combined claim].
```

## Checklist

- Start with the theme.
- Explain why the tables belong together.
- Give each table a distinct role.
- Avoid repeating the same coefficient interpretation.
- End with the combined economic interpretation.
