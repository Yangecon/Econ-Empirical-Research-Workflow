# Custom Templates

Use this folder for user-added examples.

Recommended structure:

```text
templates/custom/
  code/
    stata/
    r/
  writing/
    introductions/
    data_sections/
    empirical_strategy/
    results_sections/
    referee_responses/
```

## How agents should use this folder

Agents should inspect relevant examples before writing new project code or prose.

However:

- examples are style guides, not hard rules
- project README overrides these examples
- journal-specific instructions override these examples
- do not copy project-specific paths or confidential details blindly
- do not infer that one example applies to all projects
